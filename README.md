## Setup Instructions

1. **Download/Clone** the project files
2. **Add Media Assets** to the `assets/` folder:
   - Profile pictures
   - Slideshow images (5 images)
   - Video files
   - Audio files
   - Favicon
3. **Open** `index.html` in a web browser
4. **Test** all functionality offline

## Media Assets Required

See `assets/README.md` for detailed information about required media files and their specifications.

## Usage

1. Start at `index.html` (login page)
2. Enter any username (4+ characters) and password (6+ characters)
3. Click "Login" to access the home page
4. Navigate through different sections using the menu
5. All features work offline without server requirements
