# Media Assets

This folder contains all the media files used in the Student Portfolio website.

## Required Files:

### Images:
- `profile-pic.jpg` - Student profile picture (150x150px recommended)
- `slide1.jpg` - Slideshow image 1 (800x400px recommended)
- `slide2.jpg` - Slideshow image 2 (800x400px recommended)
- `slide3.jpg` - Slideshow image 3 (800x400px recommended)
- `slide4.jpg` - Slideshow image 4 (800x400px recommended)
- `slide5.jpg` - Slideshow image 5 (800x400px recommended)
- `hobby1.jpg` - Photography hobby image (300x200px recommended)
- `hobby2.jpg` - Music production hobby image (300x200px recommended)
- `hobby3.jpg` - Gaming hobby image (300x200px recommended)
- `career1.jpg` - Software development career image (300x200px recommended)
- `career2.jpg` - Machine learning career image (300x200px recommended)
- `career3.jpg` - Entrepreneurship career image (300x200px recommended)

### Videos:
- `instructional-video.mp4` - Instructional video for homepage (max 10MB recommended)
- `career-vision.mp4` - Career vision video for page2.html (max 10MB recommended)

### Audio:
- `music-sample.mp3` - Music sample for page1.html (max 5MB recommended)

### Icons:
- `favicon.ico` - Website favicon (16x16px or 32x32px)

## Notes:
- All images should be optimized for web use
- Videos should be in MP4 format for maximum compatibility
- Audio should be in MP3 format
- Use appropriate alt text for accessibility
- Consider using WebP format for images for better compression (with fallbacks)

## Placeholder Images:
For testing purposes, you can use placeholder images from services like:
- https://picsum.photos/ (for random images)
- https://via.placeholder.com/ (for solid color placeholders)
- https://unsplash.com/ (for high-quality stock photos)

Example usage:
- Profile picture: https://picsum.photos/150/150
- Slideshow images: https://picsum.photos/800/400
- Hobby images: https://picsum.photos/300/200
